# s21_cat & s21_grep

Мой первый учебный проект, выполненный в Школе 21.  
Нужно было реализовать упрощённые аналоги стандартных утилит Linux: `cat` и `grep`.
## О проекте

В ходе проекта необходимо было понять работу базовых утилит Bash и научиться реализовывать их на языке C, а также понять принципы структурного программирования, работы с аргументами командной строки и регулярными выражениями.

---
## Реализованные утилиты

### s21_cat

Аналог утилиты `cat`.

Поддерживаемые флаги:
- `-b` — нумерует только непустые строки
- `-e` — отображает символ `$` в конце строк (аналог `-vE`)
- `-n` — нумерует все строки
- `-s` — сжимает повторяющиеся пустые строки
- `-t` — отображает табы как `^I` (аналог `-vT`)

Расположение: `src/cat/`
Исполняемый файл: `s21_cat`

---
### s21_grep

Аналог утилиты `grep`.

Поддерживаемые флаги:

- `-e` — шаблон поиска
- `-i` — игнорирование регистра
- `-v` — инверсия совпадений
- `-c` — вывод только количества совпадений
- `-l` — вывод только имён файлов
- `-n` — вывод номеров строк
- `-h` — не выводить имена файлов
- `-s` — подавление ошибок
- `-f` — шаблоны из файла
- `-o` — вывод только совпавших частей строки

Расположение: `src/grep/`  
Исполняемый файл: `s21_grep`

## Требования к проекту

- Язык: **C**, стандарт 11
- Компилятор: `gcc`
- Стандарт: **POSIX.1-2017**
- Стиль: **Google Style**
- Форматирование: `clang-format` (14 версия)
- Сборка: через `Makefile` с целями `s21_cat`, `s21_grep`
- Соблюдение принципов структурного программирования

---
## Тестирование

Интеграционные тесты для проверки всех флагов (в виде скрипта на Bash).  Результаты сравниваются с поведением оригинальных утилит `cat` и `grep`.

## Быстрый запуск

- `s21_cat`
```bash
cd src/cat

make all      # сборка + запуск тестов
make s21_cat  # только сборка
make clean    # очистка
```
- аналогично для `s21_grep`
```bash
cd src/grep

make all  
make s21_grep  
make clean
```
